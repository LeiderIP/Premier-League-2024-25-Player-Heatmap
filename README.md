# Premier League 2024/25 — Player Heatmap Explorer

Interactive touch heatmap for every player in the 2024/25 Premier League season, built from StatsBomb event-level data.

## Features

- **Full Season** mode — all touches across all 38 matchweeks for any player
- **Single Match** mode — pick any of the 380 matches and explore any player
- **Action type filter** — All / Passes / Carries / Pressures / Shots / Duels / Defensive
- Gaussian KDE heatmap rendered in real time on HTML Canvas
- Full pitch SVG overlay with penalty areas, goals, and arcs
- **Stats panel**: total touches, matches played, avg/match, action breakdown
- **Dark / Light theme toggle**
